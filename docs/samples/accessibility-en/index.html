Accessibility statement

Conformance status

The Web Content Accessibility Guidelines (WCAG) defines requirements for designers and developers to improve accessibility for people with disabilities. It defines three levels of conformance: Level A, Level AA, and Level AAA.

The [service/website name] is fully conformant with WCAG 2.1 level AA.

Assessment approach

The [Team] assessed the accessibility of the service using:

  • WebAIM’s WAVE Evaluation tool
  • Pa11y

Compatibility with browsers

The service is designed to be compatible with the following browsers:

  • Chrome on Microsoft Windows
  • Chrome on Android
  • Edge on Microsoft Windows
  • Firefox on Microsoft Windows
  • Safari on MacOs
  • Safari on iOS

Limitations and alternatives

Despite our best efforts to ensure the accessibility of the service, there may be some limitations. Below is a description of known limitations, and potential solutions. Please contact us if you observe an issue not listed below.

[Example] The tabs do not work well with keyboard navigation. Specifically tabs do not display the focus state. This is a known issue of the gov.cy design system and will be addressed in its upcoming update.

Javascript is required for the service to function properly.

Feedback and complaints

We welcome your feedback on the accessibility of the service. Please let us know if you encounter accessibility barriers when using it:

  • Email: test@email.com

We try to respond to feedback within 30 working days.

If we do not reply, or do not fix the issue, you can send a complaint to:

  • Deputy Ministry of Research, Innovation, and Digital Policy
    134 Lemesos Avenue
    2015 Strovolos, Lefkosia
    Cyprus

Or you can email: info@dmrid.gov.cy.

If you are not satisfied with the response, you can send a complaint to the Commissioner for Administration and the Protection of Human Rights (Ombudsman).
